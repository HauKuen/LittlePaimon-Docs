::: tip
以下命令中的`(uid)`代表原神uid，如果没有给出，则会从上一次查询的uid、已绑定的私人Cookie或昵称中获取，在查询后就会自动记录uid。
:::

## 深渊战报
展示深境螺旋战报。
::: tip 提示
- 需要绑定私人Cookie才能展示使用的详细阵容。
- 数据更新有1~2小时的延迟。
  :::
- 命令：**`sy[本期|上期](uid)`**
- 别名：`sy|深渊战报|深渊信息`
- 参数：
  - 期数：可选，默认本期，可选上期
- 示例：
  - `sy100123456`
  - `sy上期`
    ::: details 深渊战报图
    ![sy](https://static.cherishmoon.fun/LittlePaimon/readme/new/sy.jpg)
    :::

- 命令：**`深渊配队`**
